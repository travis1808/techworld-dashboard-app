01

Convert .xlsx → Google Sheet

File xlsx chưa phải Google Sheet thật — cần convert để Apps Script đọc được

ℹ
Bạn đã có file techworld_ecommerce_clean.xlsx trong folder Agentic AI trên Drive. Chỉ cần mở và lưu lại dạng Google Sheets.
1
Mở file trong Google Drive
Vào Drive → folder Agentic AI → double-click vào techworld_ecommerce_clean.xlsx
2
Lưu dạng Google Sheets
Trong menu: File → Save as Google Sheets
Google sẽ tạo một bản copy mới dưới dạng .gsheet — đây mới là file chúng ta sẽ làm việc.
3
Kiểm tra tên tab Sheet
Nhìn xuống dưới cùng, tab sheet phải có tên techworld_ecommerce_clean — nếu tên khác, hãy rename đúng tên này (hoặc cập nhật biến SHEET_NAME trong Apps Script).
4
Copy Sheet ID từ URL
URL Sheet sẽ có dạng:
URL https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
Lưu lại phần [SHEET_ID] này — sẽ cần khi cấu hình Apps Script từ ngoài file.
02

Paste Apps Script vào Sheet

Script sẽ xử lý cả write (từ form) lẫn read (cho dashboard)

★
File apps_script.gs đã được tạo sẵn — chứa đầy đủ doPost (nhận data từ form) và doGet (trả data cho dashboard).
1
Mở Apps Script Editor
Trong Google Sheet: Extensions → Apps Script
2
Xóa code mặc định và paste code mới
Xóa hết nội dung file Code.gs mặc định, paste toàn bộ nội dung file apps_script.gs vào.
apps_script.gs — dòng đầu tiên sẽ trông như sau:
const SHEET_NAME = 'techworld_ecommerce_clean'; const SPREADSHEET_ID = ''; // để trống nếu script nằm trong chính Sheet
3
Lưu script
Nhấn Ctrl+S hoặc click icon lưu. Đặt tên project là TechWorld API.
03

Deploy thành Web App

Tạo URL endpoint công khai để Landing Page và Dashboard gọi vào

!
Phần này quan trọng nhất — cần set Execute as: Me và Who has access: Anyone để Landing Page có thể POST data vào.
1
Click Deploy → New deployment
Góc trên phải Apps Script editor: Deploy → New deployment
2
Chọn loại deployment
Click icon ⚙️ → chọn Web App
3
Cấu hình quyền truy cập
Description: TechWorld API v1
Execute as: Me (your email)
Who has access: Anyone
4
Lấy Web App URL
Sau khi deploy, copy URL có dạng:
URL https://script.google.com/macros/s/[SCRIPT_ID]/exec
Quan trọng: Lưu URL này lại ngay, sẽ dùng ở bước tiếp theo.
04

Gán URL vào Landing Page & Dashboard

Cập nhật 2 file HTML với Apps Script URL vừa tạo

1
Cập nhật techworld_landing.html
Mở file, tìm dòng sau và thay URL:
// Tìm dòng này (khoảng dòng 320): const APPS_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec'; // Thay bằng URL thật của bạn: const APPS_SCRIPT_URL = 'https://script.google.com/macros/s/AKfyc.../exec';
2
Cập nhật techworld_dashboard.html
Mở file dashboard, tìm và thay tương tự:
const API_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
3
Test kết nối
Mở URL này trên browser để kiểm tra:
Test [YOUR_APPS_SCRIPT_URL]?action=summary
Nếu thấy JSON với status: "success" thì kết nối thành công!
05

Mở Dashboard và kiểm tra

Toàn bộ hệ thống đã sẵn sàng

Mở techworld_landing.html → điền form → submit → kiểm tra Sheet có row mới
Mở techworld_dashboard.html → charts tự load từ Sheet
Mỗi khi có đơn mới, refresh Dashboard để thấy data cập nhật
Cowork có thể tự động refresh Dashboard theo lịch (xem hướng dẫn Cowork)
★
Muốn host online? Upload cả 3 file HTML lên GitHub Pages hoặc Netlify (free) để có URL public chia sẻ được. Apps Script URL vẫn giữ nguyên, không cần thay đổi gì.
